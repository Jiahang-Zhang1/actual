Actual ML Recommended Queries

Use these during the final demo when Prometheus is open. Grafana is the cleaner view; Prometheus is for showing the raw metric and alert expressions.

Service health and traffic

  • max(up{job="smartcat-serving"}) - serving scrape health.
  • smartcat:request_rate_5m - predict request rate.
  • smartcat:p95_latency_ms_5m - p95 latency in ms.
  • smartcat:error_rate_5m - 5xx error rate.

Model output

  • smartcat:avg_confidence_5m - average prediction confidence.
  • smartcat:low_confidence_ratio_5m - low-confidence prediction share.
  • topk(10, sum by (category_id) (increase(predicted_class_total[1h]))) - predicted category mix.

User feedback and rollout gates

  • smartcat:top1_acceptance_1h - user accepted model top-1 suggestion.
  • smartcat:top3_acceptance_1h - user chose any Top-3 suggestion.
  • count(ALERTS{alertstate="firing",alertname=~"Smartcat.*"}) - active rollout alerts.

Data quality

  • actual_data_quality_pass - latest pass/fail by data stage.
  • actual_data_quality_issue_count - issue count by data stage.
  • actual_data_quality_metric - numeric ingestion/training/drift metrics.
